这是一个示例项目，展示了Hugo Blox Builder的项目管理功能。

## 项目概述

本项目旨在展示如何使用Hugo Blox Builder创建和管理研究项目页面。项目页面支持：

- **丰富的内容格式**：Markdown、LaTeX数学公式、代码高亮
- **媒体集成**：图片、视频、交互式元素
- **社交分享**：内置分享按钮
- **SEO优化**：结构化数据和元标签
- **响应式设计**：适配所有设备

## 技术特性

### 数学公式支持

当启用数学功能时，可以编写如下表达式：

$$E = mc^2$$

### 代码示例

```python
import numpy as np
import matplotlib.pyplot as plt

def generate_data(n_samples=100):
    """生成示例数据"""
    X = np.random.randn(n_samples, 2)
    y = (X[:, 0] + X[:, 1] > 0).astype(int)
    return X, y

# 使用示例
X, y = generate_data()
print(f"生成了 {len(X)} 个样本")
```
